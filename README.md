# 🚖 Ola Ride Analytics Dashboard (Power BI)

## 📊 Project Overview
This project analyzes ride booking data to extract meaningful insights related to bookings, revenue, cancellations, and customer behavior.

---

## 🚀 Key Features
- Booking Trends Analysis
- Revenue Breakdown by Payment Methods
- Vehicle Type Performance
- Cancellation Analysis
- Ratings Insights

---

## 🛠️ Tools Used
- Power BI
- DAX
- Excel / CSV
- Data Cleaning
